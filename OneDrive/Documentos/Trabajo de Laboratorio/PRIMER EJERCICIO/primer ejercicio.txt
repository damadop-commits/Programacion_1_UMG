5. Que es un icono:

Es un simbolo grafico que aparece en la pantalla de un computador o de otros dispositivos electronicos y que presentan un programa, un sistema operativo, etc.

6. Ingrese a la carpeta de accesibilidad de windows y escriba el nombre de los iconos que aparecen y escriba su funcion:

-Lupa:Amplía la pantalla o partes de ella para facilitar la visualización de texto e imágenes.
-Narrador: es un lector de pantalla integrado que lee en voz alta lo que aparece en la pantalla, facilitando la navegación y el uso de aplicaciones a personas con discapacidad visual o dificultades para leer.
-Reconocimiento de voz de Windows: permite controlar el equipo y dictar texto con comandos de voz.
-Teclado en pantalla: permite escribir y navegar por el sistema sin un teclado físico, útil para usuarios con dificultades motoras o cuando el teclado físico falla.

8. Ingrese a la carpeta herramientas administrativas y escribir el nombre de cada icono y su funcion:

-Administracion de equipos: permiten a los administradores y usuarios avanzados gestionar diversos aspectos del sistema, como la configuración del hardware, la gestión de usuarios, la administración de servicios, la visualización de eventos, y más
-Administracion de impresion: permite gestionar la impresión, incluyendo la instalación y configuración de impresoras, la gestión de colas de impresión y la configuración de las opciones de impresión.
-Configuracion del sistema: permitir a los usuarios realizar ajustes y modificaciones en el proceso de inicio del sistema operativo
-Desfragmentar y optimizar unidades: para optimizar el rendimiento de los discos duros (HDD) y, en menor medida, las unidades de estado sólido (SSD). Desfragmenta los discos duros, reubicando los archivos para que estén más juntos y facilitar su acceso.
-Diagnostico de memoria de windows: permite probar la memoria RAM (RAM) de tu computadora para detectar posibles problemas. Si su computadora funciona lentamente, se congela, se bloquea o muestra una pantalla azul, esta herramienta puede ser útil para diagnosticar el problema. 
-Directiva de seguridad local: establecer medidas de seguridad a nivel del sistema local.
-Editor del Registro: permite modificar las configuraciones del sistema y las aplicaciones, incluyendo las políticas del sistema, las aplicaciones instaladas y los tipos de archivos que ciertas aplicaciones pueden abrir. 
-Informacion del sistema: Proporciona una vista completa del hardware, los componentes del sistema y el entorno de software de tu computadora.
-Iniciador ISCSI:una funcionalidad que forma parte de los accesorios, permite a los servidores Windows acceder a dispositivos de almacenamiento a través de la red, usando el protocolo iSCSI. 
-Liberador de espacio en disco: s una herramienta que ayuda a liberar espacio en el disco duro eliminando archivos innecesarios.
-Monitor de recursos: permite supervisar el rendimiento y el uso de recursos del ordenador, incluyendo la CPU, memoria, disco y red. 
-Monitor de rendimiento: no es un accesorio, sino una herramienta integrada de sistema que permite monitorear el rendimiento en tiempo real del equipo.
-ODBC data sources (32-bit): se encargan de la gestión de fuentes de datos ODBC (32-bit) son el Administrador de Orígenes de Datos ODBC y los archivos Odbcad32.exe.
-Origenes de datos ODBC (64-bits): herramientas administrativas que permiten a las aplicaciones acceder a bases de datos utilizando el estándar ODBC.
-Programador de tareas: es una herramienta de sistema que permite automatizar la ejecución de programas, scripts y otras tareas en momentos específicos o en respuesta a eventos.
-Servicios:  son programas que se ejecutan en un segundo plano para proporcionar funcionalidades del sistema sin la necesidad de una interfaz de usuario. 
-Servicios de Compnentes: son una herramienta para administrar y configurar componentes COM+ y DCOM, que son tecnologías utilizadas para la creación de aplicaciones distribuidas.
-Unidad de recuperacion: sirve para crear una copia de seguridad de los archivos del sistema necesarios para restaurar el equipo a su estado original.
-Visor de eventos: sirve como herramienta para registrar y visualizar eventos importantes que ocurren en el sistema operativo y en las aplicaciones.
-Windows DefenderFirewal: Funciona como un sistema de seguridad que monitoriza y controla el tráfico de red entrante y saliente para proteger tu dispositivo de accesos no autorizados y amenazas en línea.
